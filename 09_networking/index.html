Control a LED from This Website!

I worked together with Maggie. First, we followed the Random Nerd Tutorial on using a ESP32 to write and read information to and from a realtime database on Firebase. Reading through the tutorial, following the steps and using the code provided by the authors worked. However, it was a bit harder to adjust Nathan's code on the LED to make it work. We mainly used the set up from the NerdTutorial code. However, we also had to switch Firebase.set to Firebase.RTDB.set. Furthermore, we also figured out we had to add a '&' before the firebase data object in the firebase.RTDB.set() and Firebase.RTDB.get() functions.

Example: Firebase.RTDB.set(&firebaseData, "/LED_STATUS", "OFF");

After those changes the code worked and we were able to control the LED by manually changing our variable(s) in the Realtime Database Console.

Next, we used Nathan's code for ON and OFF buttons on webpages and included our own url and APi key. See the buttons below.

Trivia Game Show Simulation

We decided to implement a trivia section using the Open Trivia API. The API allows for true or false trivia questions as well as short answer questions. We chose to use the true or false questions and have two different color LEDs indicate whether the user's choice was correct or incorrect.

After attaching the APi to the page. We first used web alerts as a placeholder for the LEDs to ensure the functionality was correct before adding the output devices. We then used similar logic as the LED firebase example from class to change the statuses of our LEDs to correspond to whether the question was answered correctly or not which is similar to how game shows work.

Weather Indicators

For our weather section we used the National Weather Services API to collect realtime forecasts for Boston. We wanted to create an alert for when the weather was above a certain temperature. We used the API to collect the temperature data and compare it to a threshold that we set at 30 degrees Fahrenheit to begin with. (because it was 37 F and we wanted to see the LED turn on)

We then attached a white LED to our circuit and ESP32 that would turn on when the weather surpassed our threshold. This would be useful for attaching to doors of homes so users could prepare accordingly. However, to expand on this we replaced the LED with a more complex output device - a water pump. This has practical use in the outside world similar to automatic sprinklers where the water starts pumping water once a certain temperature is recorded.
