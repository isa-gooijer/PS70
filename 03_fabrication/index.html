KINETIC SCULPTURE!

I was inspired by Chris' Geneva drive Kinetic Sculpture and explored some of the websites he referred to on his webpage. Through his webpage I found 507movements.com which shows animations of mechanical movements and this Geneva drive simulation.

Some of my favorite animated mechanical movements from 507 Movement were:

  • Geneva Stop
  • Wheels
  • Jaw

I didn't just want to copy his sculture so I adjusted the settings on the simulator and did my own google search on how to model and build a geneva drive with a laser cutter and fusion 360. The most useful websites and/or videos I came across were Autodesk Instructables and Geneva Wheel Geometry by J.E. Johnson.

(For including joints: Geneva Gear)

Measurements

From Geneva Wheel Geometry video by J.E. Johnson Given values: a = drive crank radius n = driven slot quantity p = drive pin diameter t = allowed clearance Calculations: c = center distance = a / sin(180 / n) b = Geneva wheel radius = √ c² - a² s = slot center length = (a + b) - c w = slot width = p + t y = stop arc radius = a - ( p 1.5 ) z = stop disc radius = y - t v = clearance arc = bz / a

ARDUINO CODE:

const int A1A = 3; // define pin 3 for A-1A (PWM Speed) const int A1B = 4; // define pin 4 for A-1B (direction) void setup() { pinMode(A1A, OUTPUT); // specify these pins as outputs pinMode(A1B, OUTPUT); analogWrite(A1A, 0); // start with the motors off digitalWrite(A1B, LOW); } void loop() { // start the motor at a low speed analogWrite(A1A, 150); digitalWrite(A1B, LOW); delay(2000); // allow the motor to run for 4 seconds // stop the motor analogWrite(A1A, 0); digitalWrite(A1B, LOW); // setting both pins LOW stops the motor delay(3000); // keep the motor off for 2 seconds }
