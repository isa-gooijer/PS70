FINAL PROJECT PROPOSAL

Self-sufficient Greenhouse

What: A incubation chamber for plants with automated light, water, and nutrient regulation.

Main Purpose: Recreational. Growing herbs at home.

Other Purposes: Scientific Experiments. Growing vegetables (will have to repot the plants when they grow larger)

Functions:

  • Interface: Select a program on the interface. This can be a pre-programmed option for a certain type of herb or users can create and save their own programs with their own desired light, water and nutrition settings. Planting seeds and harvast has to be done manually.

  • Transportation: The icubation chambers and greenhouses should be easy to take apart and rebuild so they can be transported and/or stored easily.

  • Extend option: Individual Exubation Chambers can be attached to create configurations that form larger greenhouses. See below for examples (e.g., 3x2, 3x3, 3x4, etc.).
