CREATE A BOX!

Created a template for a finger jointed box that will automatically adjust the size of the fingers dependent on the dimensions of the box and the number of fingers you want per side.

Based on a youtube video by Hobbyist Maker: Fushion 360: Automatically sized finger-joints

My first iteration/design in Fushion 360:

  • 100x50x50mm
  • 5, 3, and 2 fingers for length, width and height, respectively.
  • 4.4mm material thickness
